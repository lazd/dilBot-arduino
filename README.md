# dilBot
> A loving little rover


## I2C Pinouts for AltiMU

Blue - VIN

Purple - Ground

Gray - SDA - A4

White - SCL - A5


## Sensors

A1 / D15 - RC ALEV

A2 / D16 - RC Elev

A4 - SDA (Gray)

A5 - SCL (White)

D2 - PING Left

D4 - PING Center

D7 - PING Right

D8 - Encoder Left Yellow

D9 - Encoder Left White

D10 - Encoder Right Yellow

D12 - Encoder Right White
